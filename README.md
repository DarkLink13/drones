# Drones

## _Drone Management API_

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Migrations

```bash
# generate migrations
$ npm run migration:generate <name>

# execute migrations
$ npm run migration:run

# revert migrations
$ npm run migration:revert
```

## Base URL

http://localhost:9000
